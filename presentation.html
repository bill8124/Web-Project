實作上的問題

畫圖的過程，如何呈現正在畫的圖形？

  • 再使用一層畫布作為預覽，每當游標移動，就清空預覽畫布重畫一次；游標放開後，再畫到底層的畫布。

如何畫橢圓？

  • 有些人使用arc加上scale來畫出橢圓形，但會產生變形。
  • 我們使用bezierCurveTo，利用四個點畫出一條弧線。
    上下各畫一條弧線，就構成橢圓形了。

如何預覽橡皮擦的範圍？

  • 原本想製造一個div作為橡皮擦的外框跟著滑鼠跑，但這樣canvas就抓不到mouse的event。Orz
  • 於是改成在預覽畫布上畫黑色框框，並隱藏滑鼠游標。

多人共同編輯

Firebase is OP.

Firebase

  • 註冊 → 新增 App → 撰寫 Client 端 JS → 完成
  • 大幅度簡化資料的同步：只需要 push 自己定義的 object 到 Firebase，Firebase 就會把這個 object 送到所有使用者。
  • 經由 child_added 事件處理每個由 Firebase 送來的 object。
Firebase 是一個簡化即時(Real-time)服務開發的雲端平台。

輕鬆打造聊天室

  • 我們用 Firebase 紀錄畫布的繪圖過程。
  • 同理，我們也能用 Firebase 紀錄使用者的暱稱和訊息。
  • 於是，聊天室就誕生了。

Node.js & Express & EJS

  • Node.js 是一個基於 Google V8 引擎的 JavaScript 平台，讓程式設計師輕鬆撰寫可擴充(scalable)的網路程式。
  • express 是一個非常好用的 node.js 模組，提供許多方便架設網站的功能。
  • EJS 的全名是 Embedded JavaScript，能依據不同的Data產生不同的網頁。

justfont(雲端中文字型)

以往網頁中想使用特殊字型，都須轉成圖片方能呈現，主要是技術的限制，並無法在網站中使用到真正的字型，最大的缺點就是，搜尋引擎並無法辨識圖片中的資訊；justfont 突破了中文字型的限制，並以真正的「字」呈現在網頁中，可利用網頁語法加入各種標籤，讓網站的關鍵字排名大幅提升!

趕快前往 justfont吧！
